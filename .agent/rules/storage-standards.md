# Storage Standards: Media & Media Management

## 1. 📂 Структура Storage
- **Порядок в медиа**: Использовать папки `/products/` и `/categories/` для соответствующих сущностей.
- **Именование**: Использовать понятные имена (например, `id_name.png` или `id.webp`).

## 2. ⚡ Оптимизация
- **Размеры**: Грузить только те форматы и размеры, которые оптимизированы для мобильных экранов.
- **Заглушки**: Если изображение отсутствует, всегда отображать плейсхолдер с цветом категории или нейтральным `Skeleton`.

## 3. 🖼️ Иконки и Шрифты
- Использовать системные шрифты через токены.
- Иконки: SF Symbols (iOS) и Material Icons (Android) через универсальный компонент.
