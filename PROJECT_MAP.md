# 🗺 Карта проекта: Grocery App

Этот документ является «единым источником истины» для архитектуры приложения. Он описывает все существующие экраны, их назначение и текущий статус в рамках дизайн-системы **Soft Minimalism**.

---

## 🏗 Корневая структура (Root)
| Путь | Описание | Статус |
| :--- | :--- | :--- |
| `app/_layout.tsx` | Главный Stack-навигатор, провайдеры (Auth, Theme) | [READY] |
| `app/index.tsx` | Корневой редиректор (перенаправляет на Home) | [READY] |

---

## 📱 Основная навигация (Tabs)
*Расположение: `app/(tabs)/`*

| Вкладка | Путь | Описание | Статус |
| :--- | :--- | :--- | :--- |
| **Главная** | `(index)/index.tsx` | Витрина, поиск, категории и популярные товары | [READY] |
| **Корзина** | `(cart)/index.tsx` | Список покупок, унифицированная шапка, чекаут | [UNIFIED] |
| **Профиль** | `(profile)/index.tsx` | Lite Premium кабинет, монохромное меню, бонусы | [LITE] |

---

## 🔍 Каталог и Поиск (Catalog & Discovery)
*Экраны для поиска и выбора товаров.*

| Экран | Путь | Описание |
| :--- | :--- | :--- |
| **Поиск** | `app/search.tsx` | Глобальный поиск по товарам |
| **Категория** | `app/category/[id].tsx` | Просмотр товаров в конкретной подкатегории |
| **Товар** | `app/product/[id].tsx` | Карточка товара с описанием и выбором количества |

---

## 👤 Личный кабинет (User Space)
*Все, что касается профиля, настроек и персональной истории пользователя.*

| Экран | Путь | Описание | Статус |
| :--- | :--- | :--- | :--- |
| **Профиль** | `(profile)/index.tsx` | Главный экран кабинета (вкладка в меню) | [READY] |
| **Редактирование** | `app/edit-profile.tsx` | Изменение личных данных (имя, фамилия) | [READY] |
| **Заказы** | `app/orders.tsx` | Список всех покупок пользователя | [READY] |
| **Детали заказа** | `app/order/[id].tsx` | **Статусы**, состав и трекер конкретного заказа | [IN_PROGRESS] |
| **Избранное** | `app/favorites.tsx` | Список сохраненных товаров (без скачков) | [POLISHED] |
| **Мои адреса** | `app/addresses.tsx` | Список точек доставки | [READY] |
| **Адрес** | `app/manage-address.tsx` | Форма поиска и редактирования адреса | [POLISHED] |

## 👑 Панель владельца (Admin)

| Экран | Путь | Описание |
| :--- | :--- | :--- |
| **Дашборд** | `(admin)/index.tsx` | Главная панель управления |
